Computes the element-wise logical OR of the given input tensors.

@description
Zeros are treated as `False` and non-zeros are treated as `True`.

@returns
    Output tensor, element-wise logical OR of the inputs.

@param x1
Input tensor.

@param x2
Input tensor.
