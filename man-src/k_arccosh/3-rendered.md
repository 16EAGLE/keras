Inverse hyperbolic cosine, element-wise.

@description

# Examples

```r
x <- k_convert_to_tensor(c(10, 100))
k_arccosh(x)
```

```
## tf.Tensor([2.993223 5.298292], shape=(2), dtype=float32)
```

@returns
Output tensor of same shape as x.

@param x
Input tensor.

@family numpy ops
@family ops
