__signature__
keras.ops.solve(a, b)
__doc__
Solves for `x` in the equation `a * x = b`.

Args:
    a: Input tensor.
    b: Input tensor.

Returns:
    A tensor with the same shape and dtype as `a`.

Example:

>>> a = np.array([[1, 2], [4, 5]], dtype="float32")
>>> b = np.array([[2, 4], [8, 10]], dtype="float32")
>>> keras.ops.solve(x1, x2)
array([[2, 0], [0, 2]], dtype="float32")
